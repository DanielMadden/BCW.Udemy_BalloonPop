# Balloon Pop — CLAUDE.md

## What This Is
A browser-based balloon-popping game built during the Boise Codeworks coding bootcamp. Vanilla HTML/CSS/JS with sound effects and multiple themes. Extended beyond the course requirements with extra animations and game modes.

## Stack
- Vanilla HTML / CSS / JavaScript
- No dependencies, no build step

## Files
```
index.html          # Entry point
main.js             # Game logic
style.css           # Default styles
newerstyle.css      # Updated style variant
colorNormal.css     # Normal color theme
colorCodeworks.css  # Codeworks-branded theme
*.mp3               # Sound effects (8bit, burst, guitar tracks, etc.)
*.png               # Pointer and logo assets
```

## Notes
- Open `index.html` directly in a browser — no server needed
- Bootcamp project — preserve as-is, avoid refactoring
